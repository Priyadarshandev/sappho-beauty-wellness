# Sappho Beauty & Wellness

A modern, premium demo website for a luxury beauty salon based in Patna, India.

## ✨ Features
- **Premium Aesthetics**: Minimalist design with soft pink, beige, and gold accents.
- **Service Categories**: Extensive list covering Hair Rituals, Skin Care (HydraFacial), and Spa & Relaxation.
- **Glassmorphism UI**: High-end look with translucent cards and sticky navigation.
- **Fully Responsive**: Optimized for mobile and desktop screens.
- **Smooth Animations**: Integrated with **Framer Motion** for a luxurious user experience.
- **Direct Booking**: Functional booking form and a floating WhatsApp CTA.

## 🛠 Tech Stack
- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS version)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Priyadarshandev/sappho-beauty-wellness.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
