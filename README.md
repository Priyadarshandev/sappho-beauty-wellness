# Sappho Beauty & Wellness

A premium, fully responsive salon website built for Sappho 
Beauty & Wellness, Patna — a women-led beauty studio 
specialising in Korean Hair Spa and luxury beauty treatments.

Live: https://sappho-beauty-wellness.vercel.app

---

## About the Project

Built as a complete freelance client project to drive 
appointment bookings and establish a premium online presence 
for the business. The site is live and actively used by 
the salon.

Client stats: 4.9 stars · 100+ Google reviews · Est. 2023

---

## Sections

- **Hero** — Full-screen intro with Book Appointment and 
  Explore Services CTAs
- **Services** — Three categories with live pricing:
  - Hair Rituals (Korean Hair Spa ₹1499, Designer Haircut 
    ₹599, Luxury Coloring ₹1999)
  - Skin Care (Signature HydraFacial ₹2999, Luxury Facial 
    ₹1299, Skin Brightening ₹999)
  - Spa & Relaxation (Body Polishing ₹2499, Massage Therapy 
    ₹1499, Bridal Packages — Custom)
- **About** — The Sappho story with brand values and 
  client testimonial
- **Gallery** — Instagram-integrated photo gallery
- **Booking** — Appointment form with service selection, 
  date picker, time slot, and WhatsApp booking option
- **Contact** — Address, opening hours, Google Maps embed
- **Footer** — Quick links, hours, location, social links

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React
- Vercel (deployment)

---

## Installation

**1. Clone the repository**
```bash
git clone https://github.com/Priyadarshandev/sappho-beauty-wellness.git
cd sappho-beauty-wellness
```

**2. Install dependencies**
```bash
npm install
```

**3. Run development server**
```bash
npm run dev
```
